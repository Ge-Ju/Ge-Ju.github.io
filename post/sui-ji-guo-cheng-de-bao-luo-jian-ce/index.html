实验原理

设计方案

本实验是对一个正弦波加窄带高斯噪声的信号进行包络检波，首先生成一个高斯噪声，然后对其进行窄带滤波，然后加在一个正弦信号上，得到了实验需要的信号。

得到信号后，对其进行希尔伯特变换，得到其同相分量和正交分量，算出其包络：

a=(NI(t)+bcosθ)2+(NQ(t)+bsinθ)2\mathrm{a=\sqrt{(N_{I}(t)+bcos\theta)^{2}+(N_{Q}(t)+bsin\theta)^{2}}} a=(NI​(t)+bcosθ)2+(NQ​(t)+bsinθ)2​

因为包络幅值与信噪比有关，因此设计信噪比从-20到20dB，步长为5变化，得到不同信噪比下检出的包络。已知当b很小，接近0时，包络的概率密度近似服从瑞利分布：fA(a)≈aσ2e−a22σ2,a≥0\mathbf{f}_{\mathrm{A}}(\mathbf{a})\approx\frac{\mathbf{a}}{\sigma^{2}}\mathbf{e}^{-\frac{\mathbf{a}^{2}}{2\sigma^{2}}},\mathbf{a}\geq0fA​(a)≈σ2a​e−2σ2a2​,a≥0；b很大，服从高斯分布：fA(a)≈a2πb1σe−(a−b)22σ2*⁡=a≈b12πσe−(a−b)22σ2 ,a≥0\mathbf{f}_{\mathrm{A}}(\mathbf{a})\approx\sqrt{\frac{\mathbf{a}}{2\pi\mathbf{b}}}\frac1\sigma\mathrm{e}^{-\frac{(\mathbf{a}-\mathbf{b})^2}{2\sigma^2}}\overset{\mathrm{a}\approx\mathbf{b}}{\operatorname*{=}}\frac1{\sqrt{2\pi\mathbf{\sigma}}}\mathrm{e}^{-\frac{(\mathbf{a}-\mathbf{b})^2}{2\sigma^2}}\mathrm{~,a\geq0}fA​(a)≈2πba​​σ1​e−2σ2(a−b)2​*=a≈b2πσ​1​e−2σ2(a−b)2​ ,a≥0）。因此将信噪比为-20的包络按瑞丽分布画出其概率密度图，再将其它信噪比情况下按高斯分布画出概率密度图。根据它们的交点确定判定阈值，最后根据检出包络值是否达到阈值来判断是否存在包络信号。根据定义按概率密度图与x轴围成的面积计算漏警值，然后再重复生成1000组信号计算实际漏警值，每种信噪比情况下各1000组，共8000组信号。

实现方法

首先生成高斯白噪声

% 参数设置
A = 1; % 信号的幅度
handles.A = A;
guidata(hObject, handles);
fs = 200000;  % 采样频率
f0 = 80000;%中心频率
omega0 = 2*pi*f0; % 高频调制频率
B = 400;%带宽
answer = inputdlg({'请输入噪声方差'},'window',1,{'1'});
sigma = str2double(answer{1});% 噪声方差
handles.sigma = sigma;
guidata(hObject, handles);
t = linspace(0, 1, fs); % 时间范围为0到1秒，采样fs个点
M = fs;%采样点个数

% 生成接收到的信号
theta = rand;
handles.theta = theta;
guidata(hObject, handles);
theta = theta*2*pi; % 随机相位
N = randn(size(t))/sqrt(sigma); % 生成高斯噪声

然后设计带通滤波器对其滤波，得到窄带高斯白噪声，滤波器设计如下：

%% 带通滤波器
f_low = f0 - B;  % 带通滤波器低频截止频率
f_high = f0 + B;  % 带通滤波器高频截止频率
order = 200; % 滤波器阶数
f_norm = [f_low, f_high] / (fs/2);
b = fir1(order-1, f_norm,blackman(order));

得到实验信号后，对其进行希尔伯特变换，得到同相分量和正交分量：

N = handles.N;
NI = N.*cos(omega0*t) + imag(hilbert(N)).*sin(omega0*t);
NQ = -N(1,:).*sin(omega0*t) + imag(hilbert(N)).*cos(omega0*t);

根据同相分量和正交分量算出信号的包络：

a = sqrt((NI+A*theta_c).^2 + (NQ+A*theta_s).^2);%包络

利用ksdensity函数算出有无信号时的概率密度：

% 计算概率密度
[f, Ni] = ksdensity(N);
axes(handles.axes1);
% 绘制概率密度图
plot(Ni, f, 'LineWidth', 2);
title('无信号概率密度');
xlabel('信号值');
ylabel('概率密度');

[f, Vi] = ksdensity(V);
axes(handles.axes2);
% 绘制概率密度图
plot(Vi, f, 'LineWidth', 2);
title('有信号概率密度');

ylabel('概率密度');

根据不同信噪比下，包络概率密度不同分布，画出它们的图像：

newFig = figure;
z = 0:0.01:15;
hold on
for i = 1:9
    if i<2
        fz = z/sigma.*exp(-z.^2/2/sigma);%瑞利分布
%     elseif i<5
%         fz = z/sigma.*exp(-(z.^2+b(i)^2)/2/sigma).*besselj(0,b(i)*z/sigma);%莱斯分布
    else
        fz = 1/sqrt(2*pi*sigma)*exp(-(z-b(i)).^2/2/sigma);%高斯分布
    end

end

hold off

绘出图像后，观察它们的交点，确定每种信噪比下的判决门限，根据这些判决门限计算出每种信噪比下的理论漏警率，即有信号条件下的一维概率密度曲线，求小于判决门限时的曲线面积，再求有信号全曲线下的面积，两者的比值即为漏警率。

%% 理论值
for i=1:8
    [f, Vi] = ksdensity(V(i,:));
    f_s_total = trapz(Vi, f);
    % 找到区间范围在 x 值数组中的索引
    start_index = find(Vi >= 0, 1);
    end_index = find(Vi <= VT(i), 1, 'last');
    % 提取区间内的 x 和 y 值
    x_interval = Vi(start_index:end_index);
    y_interval = f(start_index:end_index);
    % 计算区间内的面积
    f_s_miss = trapz(x_interval, y_interval);
    R(i) = f_s_miss/f_s_total;%漏警率
end

根据得到的判决门限，循环产生1000组信号进行判决，计算漏警率

%% 实际值
for j=1:8
    flag = 0;
    for i=1:1000
        %窄带信号
        theta = rand;
        theta = theta*2*pi; % 随机相位
        theta_c = cos(theta);
        theta_s = sin(theta);
        N = randn(size(t))/sqrt(sigma); % 生成高斯噪声
        N = filter(b,1,N);
        NI = N.*cos(omega0*t) + imag(hilbert(N)).*sin(omega0*t);
        NQ = -N(1,:).*sin(omega0*t) + imag(hilbert(N)).*cos(omega0*t);
        a = sqrt((NI+A(j+1)*theta_c).^2 + (NQ+A(j+1)*theta_s).^2);%包络
        [f, ai] = ksdensity(a);
        % 查找概率密度最大值及其索引
        [~, max_index] = max(f);
        % 获取最大值对应的横坐标值
        x_max = ai(max_index);
        if x_max<VT(j)
            flag = flag+1;
        end
    end
    R1(j) = flag/1000;
end

最后将理论值与实际值进行比对。
