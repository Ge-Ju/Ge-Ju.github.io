数字带通传输系统仿真

2023-12-08 13 min read # 通信原理

Ⅰ简介

背景知识简介

二进制相移键控

相移键控是利用载波的相位变化来传递数字信息，而振幅和频率保持不变。在2PSK中，通常用初始相位0和π\piπ分别表示二进制“0”和“1”。因此，2PSK信号的时域表达式为：

e2PSK(t)=Acos⁡(ωct+φn){e_{2PSK}}(t) = A\cos ({\omega _c}t + {\varphi _n}) e2PSK​(t)=Acos(ωc​t+φn​)

式中，φn\varphi _nφn​表示第n个符号的绝对相位，即：

φn=0,发送0时{\varphi _n} = 0, 发送0时 φn​=0,发送0时

φn=π,发送1时{\varphi _n} = \pi, 发送1时 φn​=π,发送1时

由于表示信号的两种码元的波形相同，极性相反，故2PSK信号一般可以表述为一个双极性全占空矩形脉冲序列与一个正弦载波相乘。

这种以载波不同相位直接去表示相应二进制数字信号的调制方式，称为二进制绝对相移方式。

2PSK调制方法如下图：

和2ASK调制原理相似，只不过基带码元是双极性不归零码，基带码元d(t)和高频载波相乘实现2PSK信号的调制。

2PSK信号的解调通常采用相干解调法，原理框图如下图所示：

在相干解调过程中，如何得到与接收信号同频同相的相干载波是个关键问题。

二进制差分相移键控

2PSK相干解调时，由于载波恢复中相位有0、π0、\pi0、π模糊性，导致解调过程出现”反向工作“现象，恢复出的数字信号”0“和”1“倒置，从而使2PSK难以实际应用。为了克服这一点，提出了魏晋至差分相移键控(2DPSK)方式。

2DPSK是利用前后相邻码元的载波相对相位变化传递数字信息，所以又称相对相移键控。假设Δφ\Delta \varphiΔφ为当前码元与前一码元的载波相位差，可定义一种数字信息与Δφ\Delta \varphiΔφ之间的关系为：

Δφ=0,表示数字信息0\Delta \varphi = 0, 表示数字信息0 Δφ=0,表示数字信息0

Δφ=π,表示数字信息1 \Delta \varphi = \pi , 表示数字信息1 Δφ=π,表示数字信息1

上述对应关系也可以反过来，因此初始参考相位不同，得到的2DPSK码也不同。

2DPSK信号产生方法：先对二进制数字基带信号进行差分编码，即把表示数字信息序列的绝对码变换成相对码，然后再根据相对码进行绝对调相，从而产生二进制差分相移键控信号。调制器键控法原理图如下：

差分码可取传号差分码或空号差分码。其中，传号差分码的编码规则为：

bn=an⊕bn−1{b_n} = {a_n} \oplus {b_{n - 1}} bn​=an​⊕bn−1​

式中，ana_nan​为绝对码，bnb_nbn​为相对码，⊕\oplus⊕为模2加。最初的bn−1b_{n-1}bn−1​可任意设置。

作业目标

实验目的

  1. 掌握数字带通传输系统的基本知识。
  2. 熟悉Matlab环境并编写函数实现功能。
  3. 熟悉实验报告的规范写作方法。

实验内容

假设载波频率为fc（单位：Hz），码元传输速率为RB（单位：Baud），码元持续时间为Ts（单位：s）

  1. 产生长度为100的随机二进制码元序列。
  2. 若fc= 10RB，画出采样率为100Sample/Ts（即100个样点/码元持续时间）的BPSK调制波形（前10个码元）及其功率谱。
  3. 相干解调时假设收发载波频率相同均为fc= 10RB，初相位均为0，画出x(t)的波形，假设低通滤波器的冲激响应为连续10个1（其余为0），画出滤波器下的y(t)及判决输出（前10个码元）。
  4. 相干解调时假设收发载波频率相同均为fc= 10RB，发端初相为0，接收端初相位为π\piπ，画出x(t)的波形，假设低通滤波器的冲激响应为连续10个1（其余为0），画出此滤波器下的y(t)及判决输出（前10个码元）。
  5. 采用DPSK及延时差分相干解调，载波频率为fc= 10RB，试画出a, b, c, d点的波形（前10个码元）。

Ⅱ基本原理/算法概述

  1. 调用randi函数产生随机二进制码元序列
  2. 调用periodogram函数计算功率谱。
  3. 使用乘法器进行调制，使用filter函数进行低通滤波。
  4. 在3的基础上改变接收端载波初相，并对调判决条件。
  5. 对信号进行2DBSK编码并调制，再根据差分相干解调特性绘出各点波形。

  2. 编程结果如下：

    clc;
clear;
close all

N = 100;%信号长度
M = 10;%截取长度
L = 100;%每个码元采样次数
fc = 10000;%载波频率（自拟）
RB = fc/10;%码元传输速率
Ts = 1/RB;%码元宽度
dt = Ts/L;%采样间隔
Fs = 1/dt;%采样频率
TotalT=M*Ts;%总时间
t=0:dt:TotalT-dt;%时间
nfft = 1000;  %fft采样点数
zb=sin(2*pi*fc*t);%载波
%%基带信号产生
data = randi([0,1],N,1);%生成随机的二进制数据
wave = data(1:M)';

fz = ones(1,L);
x = wave.*fz';%将原来wave的第一行复制L次，称为L*M的矩阵

base = reshape(x,1,L*M);%产生单极性不归零矩形脉冲波形，将刚得到的L*M矩阵，按列重新排列形成1*(L*M)的矩阵

send_signal = (base-1/2)*2;%转化成双极性不归零序列

%%BPSK基带调制   
bpsk_signal = send_signal.*zb;
%%输入信号波形
subplot(221);
plot(t,send_signal,'LineWidth',2);%绘制双极性不归零序列波形
title('双极性不归零序列波形');
axis([0,TotalT,-1.1,1.1]);
xlabel('时间/s');
ylabel('幅度');
%%BPSK调制波形
subplot(223);
plot(t,bpsk_signal,'LineWidth',1);%绘制双极性不归零序列波形
title('BPSK调制波形');
xlabel('时间/s');
ylabel('幅度');
axis([0,TotalT,-1.1,1.1]);
%%功率谱
window = boxcar(length(bpsk_signal));  %矩形窗
[psd1,f] = periodogram(bpsk_signal,window,nfft,Fs);
psd1 = psd1 / max(psd1);
subplot(2,2,4);
plot(f,psd1);
title('周期图法求功率谱');
grid on

  5. 代码如下：

    clc;
clear;
close all

N = 100;%信号长度
M = 10;%截取长度
L = 100;%每个码元采样次数
fc = 10000;%载波频率（自拟）
RB = fc/10;%码元传输速率
Ts = 1/RB;%码元宽度
dt = Ts/L;%采样间隔
Fs = 1/dt;%采样频率
TotalT=M*Ts;%总时间
t=0:dt:TotalT-dt;%时间
nfft = 1000;  %fft采样点数
zb=cos(2*pi*fc*t);%载波
%%基带信号产生
data = randi([0,1],N,1);%生成随机的二进制数据

%%BDPSK编码
bdpsk = data;
for i = 1:N
    if (i == 1)%初始bn-1设为0
        bdpsk(i) = mod(bdpsk(i) + 0, 2);
    else
        bdpsk(i) = mod(bdpsk(i) + bdpsk(i-1), 2);
    end
end
wave1 = data(1:M)';
fz = ones(1,L);
x1 = wave1.*fz';%将原来wave的第一行复制L次，称为L*M的矩阵
base = reshape(x1,1,L*M);%产生单极性不归零矩形脉冲波形，将刚得到的L*M矩阵，按列重新排列形成1*(L*M)的矩阵
send_signal = (base-1/2)*2;%转化成双极性不归零序列
%%BDPSK基带调制
wave2 = bdpsk(1:M)';
fz2 = ones(1,L);
x2 = wave2.*fz';
bdpsk_send = reshape(x2,1,L*M);
bdpsk_send = (bdpsk_send-1/2)*2;
bdpsk_signal = bdpsk_send.*zb;

%%BPSK差分相干解调
bdpsk_signal_cf = -ones(1,L*M).*zb;
bdpsk_signal_cf(1+L:L*M) = bdpsk_signal(1:L*M-L);
c = bdpsk_signal.*bdpsk_signal_cf;

%%低通滤波
h = zeros(1,L*M);
h(1:10) = 1;
d = conv(c,h);
d1 = d(1:L*M);

%%抽样判决
yo = d1;
for i=1:L*M
    if yo(i)<0
        yo(i) = 1;
    else
        yo(i) = 0;
    end
end
%%输入信号波形
subplot(421);
plot(t,send_signal,'LineWidth',2);%绘制双极性不归零序列波形
title('基带信号波形');
axis([0,TotalT,-1.1,1.1]);
xlabel('时间/s');
ylabel('幅度');

Ⅴ结论

  1. 仿真调用randi函数实现了产生随机二进制码元序列
  2. 仿真调用periodogram函数实现了计算功率谱。
  3. 仿真使用乘法器进行调制，使用filter函数，实现了低通滤波。
  4. 仿真在3的基础上改变接收端载波初相，并对调判决条件，实现了题目要求。
  5. 仿真对信号进行2DBSK编码并调制，再根据差分相干解调特性绘出各点波形。并观察到解调信号在经过相乘器后，得到的信号与基带信号相反，于是在抽样判决中反向判决，实现解调出基带信号。
